# Outlier Detection Using Z-Score and IQR

This project explores data preprocessing techniques by detecting outliers using the Z-Score and Interquartile Range (IQR) methods. The analysis was performed using Python, Pandas, NumPy, Seaborn, and Matplotlib.

## Features

Statistical outlier detection using Z-Score.
Outlier detection using the IQR method.
 Comparison of detected outliers across features.
 Visualization of data distributions using box plots.
Data preprocessing techniques commonly used in machine learning workflows.

## Technologies Used

 Python
 Pandas
 NumPy
Seaborn
Matplotlib

## Learning Outcomes

Data cleaning and preprocessing.
 Exploratory Data Analysis (EDA).
 Statistical analysis of datasets.
 Data visualization and interpretation.
